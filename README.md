
## Description

REST API Backend for the quicksurvey app.
Technologies used:

[NestJS](https://github.com/nestjs/nest) \
[TypeORM](https://typeorm.io/) \
[ClassValidator Library](https://github.com/typestack/class-validator) 

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
